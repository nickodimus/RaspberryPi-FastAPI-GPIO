Get started with your GPIO Device

This site was created to help program a raspberry pi running a headless Ubuntu 23.10 Server. The Controller Pages dropdown contains some useful pages to control the GPIO states with the GPIO Control Interface page and persist these changes across reboots by following the instructions on the Initialize GPIOs page. There is also an embedded terminal in the Terminal page for directly interacting with the computer OS running this website. There is also a nodeRed server monitoring the Solar Power Plant and quite a few other small odds and ends specific to my use case.
